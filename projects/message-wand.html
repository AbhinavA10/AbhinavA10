This was a project started by my teacher that I had finished with my own idea. Waving the wand back and forth displays a message as it is swung. The length of a swing is determined by a movable piece in front of an IR sensor. Using the average of the first few swings, the micro-controller determines the endpoints of a user's swings, and then displays a message accordingly.
